# AGENTS

## 語言與風格
- 回覆使用繁體中文。
- 變更說明與 CHANGELOG 需面向使用者、避免內部細節。

## 專案概覽
- Vite + React 專案，提供 Markdown 編輯與 PDF 匯出。
- PDF 產生邏輯位於 `src/utils/pdf.ts`。

## 常用指令
- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run preview`

## 版本與發佈
- 版本號位於 `package.json` 與 `package-lock.json`。
- 變更需同步更新 `CHANGELOG.md`，格式沿用現有結構。
- 版本標籤採用 `vX.Y.Z`。
