# Changelog

所有重要變更將記錄在此檔案。

## [1.1.0] - 2026-02-12

### Added

- 在支援的瀏覽器中，可透過儲存對話框直接選擇 PDF 儲存位置

### Changed

- 匯出按鈕新增「匯出中」狀態，避免重複觸發下載
- 使用者取消儲存時會安全中止；若瀏覽器不支援則回退原本下載方式
- README 補充 PDF 匯出與儲存位置選擇的使用說明
- 新增 AGENTS.md 專案協作規範文件

## [1.0.2] - 2026-02-05

### Changed

- 強化整體 UI 的紙感色調與排版質感
- 更新編輯器與預覽的字體與互動樣式
- 更新預設範例中的 GitHub 連結

## [1.0.1] - 2026-02-05

### Changed

- 更新 jspdf 與 dompurify 相依版本
- 改善 PDF 產生設定的型別相容性

## [1.0.0] - 2026-02-05

### Added

- Markdown 編輯器與即時預覽
- GFM 語法支援（表格、刪除線、任務清單）
- PDF 匯出與自訂檔名
